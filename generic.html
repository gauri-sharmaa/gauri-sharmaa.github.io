Acid Mine Drainage Visualization

The project leverages Google Earth Engine and satellite imagery to remotely monitor and send chemicals for environmental management. By utilizing satellite data, the system tracks and assesses chemical usage and distribution across various regions. This approach provides an efficient, real-time solution for monitoring environmental impacts, ensuring that chemical applications are optimized and aligned with sustainable practices. Through this technology, the project aims to enhance the accuracy and effectiveness of remote chemical management while reducing risks to ecosystems.

EnGarde

En Garde is a web platform designed to streamline the fencing recruitment process, bridging the gap between international athletes and U.S. collegiate coaches. Inspired by team member's challenges in the recruiting process, the platform allows fencers to create profiles showcasing their skills, rankings, and practice footage, while coaches gain access to a centralized system for evaluating prospects. Built using Django, Selenium, and Chart.js, En Garde tackles issues like data inconsistency in fencing records through web scraping and analytics. Future plans include expanding to other sports, developing advanced statistical models, and improving accessibility for underrepresented athletes.

VisualEase

VisualEase is an innovative memory training web application designed to enhance active recall through AI-generated images and an interactive flashcard interface. Utilizing GROQ and the Flux.1 API, the app efficiently segments text files and rapidly generates images to facilitate learning. The seamless and responsive front-end experience is powered by LLAMA3, Node.js, and Flask, ensuring that users can easily engage with the material and improve their memory retention.

Inklusive

Inklusive is a web application developed to detect letter reversals, a common symptom of dyslexia, in handwriting. Leading a team of five, I utilized HTML and CSS to build the application, which is hosted on Django. By employing Keras OCR with multiple models tailored for different text sizes, we harnessed Python TensorFlow to process handwriting on a large scale. I played a pivotal role in conceptualizing the project’s purpose and assisting in the development of the machine learning model.
